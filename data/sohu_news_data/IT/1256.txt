
　　在国内， UNIX 随着 IBM 、 HP 、曙光、太极等计算机厂商的小型机、 PC 服务器等主机产品广泛深入到各行各业，而得到了广泛的使用
　　。在金融、电信行业的核心业务系统领域，特别是在实时的帐务处理中的应用， UNIX 几乎占据了绝对优势。但是，在 Linux 和 Windows 
的竞争压力下，在 2005 年， UNIX 还是感受到了极大的市场压力。
　　电信金融行业仍为 UNIX 最大的主顾
　　一直以来，电信和金融行业对系统的安全性和稳定性都有较高的要求，所以在稳定性和安全性的表现上都很突出的 UNIX ，一直以来都是这些行业的首选。在 
2005 年，电信和金融行业仍然是国内 UNIX 市场的主要客户，计世资讯（ CCW Research ）相关报告数据显示，电信和金融行业中应用 UNIX 
的分别占据 24.7 ％和 22.7 
％的份额，这些系统主要用在电信、金融行业的核心业务主机上，包括前置服务器、应用服务器、数据服务器等高密度数据处理的系统环境，这些高端的应用领域，仍然是 UNIX 
占据主要地位。
　　UNIX 稳定性、安全性受客户首肯
　　UNIX 产品的稳定性、安全性一直以来都获得了绝大多数用户的首肯，成为用户选择 UNIX 的主要原因。计世资讯（ CCW Research 
）相关报告显示，分别由 86% 的客户对稳定性感到满意，而 50% 的客户对 UNIX 的安全性很满意，这方面也是 UNIX 产品的突出优势。稳定性得益于 
UNIX 的系统架构，其设计思想目前仍然不落后，在安全性方面，虽然 UNIX 存在着被不法入侵者攻击的种种机会和漏洞，但与 Windows 
相比，它仍然是相当安全可靠的系统，从用户反应来看， UNIX 的安全性在相当程度上得益于 UNIX 环境的病毒的稀少。就性能而言，用户反映 UNIX 
仍然是对硬件条件要求最低，运行效率最高的操作系统。
　　Windows 、 Linux 夹击 UNIX
　　从目前市场发展趋势看， Windows 和 Linux 是 UNIX 面临的两个强劲对手。 Linux 从出现到现今，只经历了短短 9 
年的时间，但其发展速度很快，这与它的开放性和优良的性能是密不可分的，特别重要的是，其发展的动力不仅来自众多的厂家，还有遍布全球、为数众多的 Linux 
热心者、爱好者，经过几年的努力， Linux 系统在各行业领域都取得广泛的应用和进展。
　　Windows 则是 UNIX 另一个强劲的对手。 Microsoft 已凭借 Windows 垄断着全球桌面操作系统市场 90% 以上的份额， 
Windows 
系统不仅在个人操作系统中占有绝对优势，同时在网络操作系统和企业操作系统市场中，也是具有非常强劲的力量。这类操作系统在整个局域网配置中也是目前最常见的，但由于它对服务器的硬件要求较高，且稳定性能不是很高，所以 
Microsoft 的网络操作系统，目前仍然只是用在中低档服务器中。
　　长远而言，向开放架构迁移，是不可变更的趋势，但非一朝一夕所能完成。计世资讯（ CCW Research 
）研究认为，在金融、电信等行业的核心业务系统中， UNIX 
系统的核心地位，未来五年内还无可替代。但向开放架构迁移，是未来五年内不可变更的趋势，随着用户逐渐认可了其他操作系统的稳定性和安全性， UNIX 
将会遇到全面的冲击。
　　UNIX 将在中国首次出现负增长
　　计世资讯 (CCW Research) 研究显示，由于对手的强劲发展和自身的缺欠所致，到 2007 年， UNIX 
在中国将会出现首次负增长，这在几十年来是首次出现的现象。
　　UNIX 出现负增长，将产生重要并深远的影响，可以说这并非仅仅是销售收入的减少，而是整个产业链的重组和变迁。
　　中国的大型企业和政府科研机构用户，一直是 UNIX 的主要客户群，但是在中国市场，大型企业和政府科研机构的数量不到总用户数的 1 ％，超过 300 
万的中小企业，是 IT 产品更庞大的市场。 2005 年，中国中小企业信息化建设增长迅速，是整个 IT 市场中最重要的亮点，吸引着产业界的广泛关注和重视，而 
UNIX ，在满足中小企业用户需求方面还比较欠缺。 UNIX 的发展，离不开基于 UNIX 进行应用系统开发的 SI 和软件开发商的支持， UNIX 
出现负增长，会让整个 UNIX 产业链失去成长的动力，随着越来越多原 UNIX 服务商，推出基于 Windows 或 Linux 
的应用软件或解决方案，将导致整个产业链的重组和变迁，把 UNIX 带入真正的困境。
